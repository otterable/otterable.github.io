Install

First step is to install HUGO on your specific OS. If you are new to HUGO you can read the HUGO Docs. These docs are great and updated to start with HUGO installation.

Important: Since we are using SCSS make sure you have the extended version of HUGO installed on your system.

Deployment

There is no need to add the theme we have added theme to the hugo structure itself. Once you extract the zip in desired location just run: hugo server or hugo server --disableFastRender to start the HUGO server.

I would recommend checking the Deployment Methods page on HUGO website.

* We highly recommend using the Netlify service for deployment. All you need to do is create a new private repository on GitHub or GitLab. Upload the theme to the repository and link your repo to Netlify. Please check this link with the step by step guidelines.

Posts

To create a new post, you can create a new markdown file inside the content/posts directory.

You can set the title, description, tags, featured or not, and the post image.

The featured key is to mark the post as a featured post. This will add a simple star icon (☆) to the right corner of the post card.

Add post images to static/images directory.

Disqus Comments

Aditu Theme comes with Disqus comments enabled.

Open config.toml file, and change the anvod-studio-aditu value on line 110 with your Disqus account shortname.

That’s all you need to setup Disqus from the theme side. If you get any issue regarding that comments are unable to load. First, make sure you have registered your website with Disqus (Step 1).

And also check Disqus troubleshooting guide if you still have issues.

Google Analytics

To integrate Google Analytics, open config.toml, and add your Google Analytics identifier.

Update favicon

You can find the current favicon (favicon.ico) inside static directory, just replace it with your new favicon.
